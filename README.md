# 🚗 Car Price Prediction using Machine Learning

This project is a machine learning-based system that predicts the selling price of used cars based on various features. It includes data preprocessing, model training, evaluation, and deployment using a web application.

---

## 📊 Dataset

- **Size:** 301 used car records  
- **Features:** Car Name, Year, Present Price, Kilometers Driven, Fuel Type, Seller Type, Transmission, Owner  
- **Target:** Selling Price  

---

## ⚙️ Methodology

1. Data loading and exploration  
2. Data visualization (distribution plots, correlation heatmap)  
3. Data preprocessing and feature encoding  
4. Train-test split (80/20)  
5. Model training using regression algorithms  
6. Model evaluation using R², MAE, RMSE  

---

## 🤖 Models Used

- Linear Regression  
- Lasso Regression  
- Random Forest Regressor  

---

## 📈 Results

- Random Forest achieved the best performance with high accuracy  
- Present price is the most important feature affecting prediction  
- Model effectively captures relationships between features and selling price  

---

## 🌐 Web Application

- Built using **Streamlit**  
- Takes user inputs like year, price, fuel type, etc.  
- Predicts car resale value in real-time  
- Provides insights like depreciation and value comparison  

---

## 🛠️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib / Seaborn  
- Streamlit  

---

## 🚀 Deployment

- Model saved using joblib  
- Deployed on Streamlit Cloud  
- Accessible via web browser  

---

## 🔍 Key Insights

- Present price has the highest impact on resale value  
- Random Forest performs better than linear models  
- Feature engineering improves prediction accuracy
