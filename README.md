# 💳 Credit Card Fraud Detection using Machine Learning

This project focuses on detecting fraudulent credit card transactions using machine learning techniques. It handles highly imbalanced data and applies classification models to accurately identify fraud cases.

---

## 📊 Dataset

- **Source:** Public credit card transaction dataset  
- **Features:** Anonymized features (V1–V28), Time, Amount  
- **Target:** Class (0 = Legitimate, 1 = Fraud)  

---

## ⚙️ Methodology

1. Data loading and exploration  
2. Data preprocessing and cleaning  
3. Handling imbalanced data using resampling techniques  
4. Feature engineering and scaling  
5. Train-test split  
6. Model training using classification algorithms  
7. Model evaluation using accuracy, confusion matrix  

---

## 🤖 Models Used

- Logistic Regression  
- Random Forest Classifier  

---

## 📈 Results

The model achieves strong performance in detecting fraudulent transactions. Handling imbalanced data significantly improves model accuracy and reliability.

## 🔍 Key Insights

- Fraud datasets are highly imbalanced and require special handling  
- Feature engineering plays a crucial role in improving performance  
- Ensemble models like Random Forest provide better accuracy
